# SmartInbox

An AI-powered smart email client built with React and FastAPI. SmartInbox connects to your Gmail account and brings intelligence to your inbox — detecting suspicious emails, summarizing long threads, flagging sponsors and job opportunities, and keeping your most important contacts organised.

## Live Demo

https://smart-inbox-weld.vercel.app

## Features

- **Gmail OAuth 2.0** — Secure login with your Google account
- **Smart Email Scanner** — Detects suspicious, sponsored, career, and important emails with animated scan buttons and color coded badges
- **AI Email Summaries** — Summarizes any email into 2 lines using Llama 3.1 via Groq API
- **Full Inbox Scan** — Scan all emails at once with a live progress UI
- **Archive with Reminders** — Archive emails and set date based reminders with snooze support
- **Categories** — Create custom categories and add important email addresses that get flagged automatically
- **Liquid Glass UI** — Glassmorphism inspired design with smooth page transitions and animations
- **Dark Mode** — Full dark and light mode support

## Tech Stack

**Frontend**
- React 18 with Vite
- Framer Motion for animations and page transitions
- React Router for navigation
- Axios for API calls
- Lucide React for icons

**Backend**
- FastAPI with Python 3.11
- SQLAlchemy with SQLite (async)
- Google Gmail API for email access
- Groq API with Llama 3.1 for AI summaries
- APScheduler for reminder scheduling

**Hosting**
- Frontend on Vercel
- Backend on Render

## Architecture

The app follows a clean separation between frontend and backend. The React frontend communicates with the FastAPI backend via REST API. Gmail emails are fetched server side using the Gmail API with OAuth 2.0 tokens stored in a SQLite database. Email classification is done using keyword analysis in the backend classifier service. AI summaries are generated by sending email content to the Groq API and returning a 2 sentence summary.
